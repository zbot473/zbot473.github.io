Introduction

When you open your python installation, you should have the interpreter in front of you. If you press enter you will see something like this: 
>>>
If you try typing something that isn't a function or incorrect syntax, you will get something like this: 
>>>
thisissomethingwrong
            Traceback (most recent call last):
              File "<pyshell#1>", line 1, in <module>
                thisissomethingwrong
            NameError: name 'thisissomethingwrong' is not defined

However, if we use something that is a function, then it will show something like this: 
>>> 
print
(
"This will work"
) 
This will work
Let's break that down. 
print
is a builtin function. Whatever value we pass into the 
print
function in between the two parentheses will be printed out in 
blue
text. For example, we passed in a 
"string"
, and the function returned it. We could also pass in a number. For example: 
>>> 
print
(
42
)
42
Note:
The colors and font may be different in your python installation.

// #Divide and only keep the integer part: 5//3 = 1

Strings

Strings are sequences of characters. A string is enclosed is enclosed in two sets of single or double quotes like 
'this'
or 
"this"
. They can contain pretty much anything. However, they cannot be changed. You can output strings by using 
print
. What's intresting is that you can add two strings together. For example: 
>>> 
print
(
"string1"
+
"string2"
) 
string1string2
One thing to know is that you need to manually add a space between two strings. 
>>> 
print
(
"string1 "
+
"string2"
) 
string1 string2
We can also multiply strings like this: 
>>> 
print
(
"hi "
* 10
) 
hi hi hi hi hi hi hi hi hi hi
To convert numbers to string use 
str
. Here's how that works: 
>>> 
print
(
"The number is: "
+ 
str
(1234)) 
The number is: 1234
To find the length of strings, you can use 
len
. 
>>> 
print
(
len
(
"abcdefghijklmnopqrstuvwxyz"
)) 
26
